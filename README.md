# Job Portal Authentication System

This is a simple Job Portal Authentication project built using Firebase.

## 🚀 Features
- Role selection (Employer / Jobseeker)
- Phone number authentication using Firebase OTP
- Google reCAPTCHA integration
- Real-time authentication handling

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Firebase Authentication
- Firebase Firestore

## 📌 How it Works
1. User selects role (Employer or Jobseeker)
2. Enters phone number
3. Receives OTP (via Firebase)
4. Verifies OTP
5. User gets authenticated

## ⚠️ Note
- This project uses Firebase test phone numbers for authentication.
- Real OTP requires Firebase billing to be enabled.
